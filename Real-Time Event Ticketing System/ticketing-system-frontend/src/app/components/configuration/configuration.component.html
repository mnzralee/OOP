Configuration Descriptions

  • Total Tickets: The total number of tickets available for the vendor.
  • Ticket Release Rate: The rate at which tickets are released by vendors to the ticket pool.
  • Customer Retrieval Rate: The rate at which customers can retrieve their tickets.
  • Max Ticket Capacity: The maximum number of tickets that can be held in the ticket pool at any given time.
